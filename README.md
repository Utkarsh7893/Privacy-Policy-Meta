## 🔐 What is a Privacy Policy?

A **privacy policy** is a document that explains how an application handles your data. Here’s a simple breakdown:

* 📊 **What data is collected**
  (e.g., name, email, usage data)

* 🎯 **Why the data is collected**
  (e.g., to improve services, provide features)

* 🔄 **How the data is used**
  (e.g., analytics, authentication)

* 🤝 **Whether data is shared**
  (e.g., with third-party services)

* 🛡️ **How your data is protected**
  (security measures in place)

* ⏳ **How long data is stored**

* 🚪 **Your rights as a user**
  (e.g., access, delete, or modify your data)

👉 In short: it helps you understand what happens to your data when you use an app.
